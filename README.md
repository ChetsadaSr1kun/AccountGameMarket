# AccountGameMarket

ตลาดซื้อขายไอดีเกม (GameMarket) - โปรเจกต์ไฟนอลที่ผ่านการจัดระเบียบไฟล์เรียบร้อยแล้ว

## โครงสร้างโปรเจกต์ (Project Structure)
```text
GameMarket/
├── assets/
│   ├── css/
│   │   └── style.css            (ไฟล์สไตล์ CSS ที่แยกออกมา)
│   ├── js/
│   │   └── script.js            (ไฟล์ระบบ JavaScript ที่แยกออกมา)
│   └── images/                  (โฟลเดอร์เก็บไอคอนและแบนเนอร์ของเกมทั้งหมด)
│       └── valorant/            (รูปภาพสกรีนช็อตเฉพาะเกม Valorant)
├── documentation/
│   ├── 241.pdf                  (ไฟล์รายงาน PDF)
│   └── Untitled Diagram-Page-9.drawio.png (ไฟล์ Diagram งานออกแบบ)
└── GameMarket.html              (ไฟล์ HTML หลักที่เชื่อมลิงก์ภายนอกเรียบร้อยแล้ว)
```
